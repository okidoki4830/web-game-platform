# Web Game Platform – Flappy Planes

## Overview
This project is a browser-based game platform developed as part of a university team project at Florida State University. 
The platform includes multiple games accessible through a central hub.

I independently implemented a Flappy Bird–style game using HTML, CSS, and JavaScript, including game physics, obstacle generation, and score tracking.

## Demo
Play the game here:

https://okidoki4830.github.io/web-game-platform/

## My Contribution
I independently developed the **Flappy Planes** game including:

- Character movement and gravity-based physics
- Randomized pipe obstacle generation (different heights and gaps each run)
- Collision detection system
- Scoreboard tracking player progress
- Game restart functionality

## Team Contribution
Other team members implemented:

- Tic Tac Toe game
- Main game hub interface
